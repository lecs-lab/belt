Projects

Build practical, usable language technology using either our example data or your own language data. These projects will assume you are familiar with the skills listed above.

Spellchecker
Design an editor that reports misspelled words, lets users add new words to the dictionary, and offers spelling suggestions.
Predictive Text
Build an engine that predicts the next word as the user is typing, using a simple n-gram model.
Word Games
Design simple word games in your language for fun and language learning.
Automatic Inflection
Build a system to automatically perform morphological inflection.
